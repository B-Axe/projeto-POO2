### Sobre o projeto: 
Esse projeto é um simulador de trânsito, desenvolvido em python. Ele serve para analisarmos a melhor entre 2 estratégias para andar no trânsito, com a estratégia de frenagem suave na estrada 1 e frenagem bruta na estrada 2. Mostrando tanto média de gasolina gasta e distância percorrida, para analisarmos.
